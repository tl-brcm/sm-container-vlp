For knowing status of pods in namespace <namespace>, use -> kubectl get pods -n <namespace> (Where <namespace> is variable where the POD is deployed)
For live status of pods use -> watch kubectl get pods -n <namespace>


For logs use -> kubectl logs [pod_name] -c [container_name] -n <namespace>
EG: -> kubectl logs <release name>-siteminder-access-gateway-<xxxx> -c access-gateway -n <namespace> (where <xxxx> is variable which changes after every deployment)


[container_name] of casso-access-gateway-x pods are: 
1. access-gateway
2. runtime-config-retriever (sidecar container)
3. agmetricsexporter (sidecar container)
4. config-retriever (init-container)
