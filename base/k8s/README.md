# base/k8s
This is a sample subdirectory that is used to keep
a sample self-signed certificate and its private key
file. This is used as the default for DOMAIN, CERTFILE, KEYFILE
used in the base/env.shlib

* k8s.pem -- a self-signed certificate for "k8s.demo"
* k8s.key -- the corresponding private key file for k8s.pem
